# Khushi AI — Project Roadmap

This document outlines the short-term and long-term milestones for **Khushi AI**.

---

## 🚀 Version 1.0.x — Stable Open-Source Release (Current Focus)
- [x] **PySide6 Dashboard UI**: Modern responsive dashboard with settings panels.
- [x] **Local REST/WebSocket Servers**: FastAPI pipelines and WebSocket stream connectors.
- [x] **Core Intent/Skill Managers**: Robust algebraic calculator, notes skills, screenshot skills.
- [x] **Diagnostics & Robustness**: Complete crash recovery checkpoints, startup watchdog, and timeline profiling.
- [x] **Standard open-source audits**: Reorganized structures, templates, licenses, example files.

---

## 🔮 Version 1.1.x — Enhanced Automation & Vision
- [ ] **Advanced OCR overlay visualizers**: Draw overlays on active screen panels during automated tasks.
- [ ] **Dynamic Plugin Store**: Sandbox registry updates to download third-party plugins directly from verified remote lists.
- [ ] **Automated Keyring Management**: Encrypted storage for external API tokens (e.g. OpenAI/Anthropic/Weather services).

---

## 📱 Version 1.2.x — Cross-Device Synchronization
- [ ] **Local Network Discovery**: Automatically discover pairing endpoints on local subnets using ZeroConf / mDNS.
- [ ] **React Native Mobile App Release**: Package and release stable builds for Android/iOS.
- [ ] **Encrypted LAN streams**: AES-256 encrypted WebSocket transport between desktop core and paired phones.

---

## 🧠 Version 2.0.x — Cognitive Expansion
- [ ] **Local LLM Orchestration**: Direct inference using GGUF format (via llama.cpp bindings) for complete offline conversations.
- [ ] **Autonomous Agent Routines**: Multi-agent task planning (using `planner/`) where agents collaborate to achieve long-running user targets.
