# Security Policy

## Supported Versions

We actively maintain and provide security patches for the following versions of Khushi AI:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

---

## Reporting a Vulnerability

We take the security of Khushi AI seriously. If you find a security vulnerability, please do **not** report it via public GitHub issues. Instead, follow the process below:

1. **Email us directly**: Send a detailed description of the vulnerability to **security@khushi-ai.org** (or your personal email).
2. **Include details**: To help us verify and patch the issue quickly, please include:
   - Steps to reproduce the vulnerability (including proof of concept script/steps).
   - The environment/operating system versions.
   - The potential impact.
3. **Acknowledgment**: We will acknowledge receipt of your report within 48 hours and work with you to patch the issue.
4. **Disclosure**: Once a patch is released, we will coordinate public disclosure with you to ensure users are protected.
